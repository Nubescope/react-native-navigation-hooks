Before you start

React Native Navigation Hooks is a module, dependent on and intended to be used alongside React Native Navigation, so some experience with it and knowledge of core concepts is required. If you feel the need, you can start with their getting started page, and then come back here when you're ready.

If you want to dig right into it, start with installing the library. If you're just looking around, we suggest checking out usage first.

## `withNavigationProvider`

Creates a higher-order React component class that wraps your component with the [NavigationProvider](contexts#NavigationProvider) and sets the provider value with **React Native Navigation** injected props like `componentId`.

```tsx
import { Navigation } from 'react-native-navigation'
import { withNavigationProvider } from 'react-native-navigation-hooks'
import { MyScreenComponent } from '~/screens/MyScreenComponent'

Navigation.registerComponent('MyScreenComponent', () => {
  return withNavigationProvider(MyScreenComponent)
})
```
